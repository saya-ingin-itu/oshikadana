# Googleフォーム作成手順（作業時間 約2分）

1. https://forms.google.com を開く → 「空白のフォーム」
2. タイトル: `オシカダナ 事前登録`
3. 説明: `リリース時にメールで1通お知らせします。それ以外には使いません。`
4. 質問1つだけ作る:
   - 種類: 記述式（短文）
   - 質問文: `メールアドレス`
   - 必須: ON
   - 右下⋮ → 「回答の検証」→「テキスト」「メールアドレス」を選択
5. 設定タブ → 「回答を1回に制限する」はOFFのまま（ログイン必須になるため）
6. 右上「送信」→ `< >`（埋め込み）タブ → HTMLをコピー
7. コピーしたHTML内の `src="https://docs.google.com/forms/d/e/.../viewform?embedded=true"` のURL部分をClaudeに渡す
   → Claudeが docs/index.html の iframe に反映して再デプロイする

回答数の確認: フォーム編集画面の「回答」タブ。判定日にスクショを共有。
